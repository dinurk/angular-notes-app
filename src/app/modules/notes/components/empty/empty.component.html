В данном окне будут отображаться:

  • Форма добавления заметки - переход нажатием кнопки "Добавить"
  • Окно просмотра заметки - переход при нажатии на заметку
  • Форма изменения заметки - переход из окна просмотра заметки
  • Заметки, отображаемые в левой половине экрана можно группировать нажатием клавиши ctrl
  • Напоминаия можно добавить на вкладке "Напоминания", когда есть хотя бы одна заметка
  • Установленные напоминания отображаются в календаре на вкладке "Напоминаия"
